
# 28. Implement strStr()
### [Back ⬅️](README.md)

## **Python 3** solution

```py
class Solution:
    def strStr(self, haystack: str, needle: str) -> int:
        if len(haystack) < len(needle):
            return -1

        if len(needle) == 0:
            return 0

        for i in range(len(haystack) - len(needle) + 1):
            if haystack[i] == needle[0]:
                j = 0

                while j < len(needle):
                    if haystack[i+j] != needle[j]:
                        break
                    else:
                        j += 1

                if j == len(needle):
                    return i

        return -1
```
